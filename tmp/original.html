Email Settings · EmailJS

Sending memos by email uses EmailJS — a free service (200 emails/month) that works directly from this page with no backend required. Configure it once and your settings are saved in this browser.

One-Time Setup (5 min)

  1. Create a free account at emailjs.com
  2. Go to Email Services → connect Gmail, Outlook, etc.
  3. Go to Email Templates → create a new template
  4. Set To Email field to: {{to_email}}
  5. Add any body text you want — all memo fields are available as variables
  6. Go to Account → API Keys and copy your Public Key
